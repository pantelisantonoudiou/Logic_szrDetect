# Logic_szrDetect
Semi-automated seizure detection toolbox. 


## How to Use

### A) Data preparation and batch predictions
    python main_process.py

### B) User Verification
    python get_path.py
    python main_gui.py

## Dependancies
Numpy
Scipy
Included adi toolbox from Jim Hokanson https://github.com/JimHokanson/adinstruments_sdk_python
